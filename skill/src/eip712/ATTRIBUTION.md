# Vendored: casper-eip-712 (TypeScript)

The files in this directory are vendored from the official
**`casper-ecosystem/casper-eip-712`** repository (`js/src`, package
`@casper-ecosystem/casper-eip-712` v1.2.1), because that package is not
published to npm.

It is used unmodified to build the `TransferAuthorization` EIP-712 digest so our
signatures are byte-compatible with the Casper x402 facilitator's verifier.

Upstream: https://github.com/casper-ecosystem/casper-eip-712
License: see the upstream repository.
